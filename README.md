# Voting Dapp
> Simple application for holding an election between two candidates(can be modified for more).

*	Using a blockchain like ethereum facilitates the fairness in elections by taking advantage of it's decentralized nature and enforcing fairness which is a liability in centralised voting.


* Done with the help of truffle development framework including web.js library,mocha testing framework,chai assertion library,etc.




## Technologies
* node    - version v16.1.0
* npm     - version 7.11.12
* Truffle - version v5.1.62
* MetaMask- version 9.4.0
